ClickSend SMS Woo Integration helps to send transactions & promotional sms to wooCommerce store owners.

== Description ==
This plugin helps to integrate [ClickSend SMS](https://www.clicksend.com/) with WooCommerce for sending orders related updates. Shop owner can also send the sms notication manually.

== Installation ==
* [Sign Up](https://dashboard.clicksend.com/signup) for a free ClickSend account.
* Copy your API key from the [API Credentials](https://dashboard.clicksend.com/#/account/subaccount) area.
* download this plugin
* upload in plugin directory
* activate it
* a settings menu is generated on plugin activation
* fill up the credentials on the plugin settings page

== Frequently Asked Questions ==
= Is it free plugin? =
Yes this plugin is completely free but you must have an account with [clicksend.com](https://www.clicksend.com) to send sms. 

= Is it official plugin by clicksend.com? =
Yes it is official plugin by clicksend.com.

= Can I send promotional sms campaign with dynamic fields? =
Yes you can create dynamic templates using placeholders e.g first_name, last_name, order_status, order_id. And send SMS in bulk to your customers.

= Does this plugin send SMS automatically when customer place the order? =
Yes by default this settings will be enable and you can control the SMS template via automation tab.

= Can I update the automatic templates? =
Yes you can update each status templates, even if you have any customised wooCommerce status that you can also update.
